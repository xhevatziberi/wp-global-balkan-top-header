A top header plugin

== Description ==

A plugin to display a top header for all Global Balkan sites.

== Installation ==

Installing "Global Balkan Top Header" can be done either by searching for "Global Balkan Top Header" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
